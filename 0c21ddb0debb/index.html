高斯积分的求解简明笔记

本文最后更新于 2021-11-11T23:15:03+08:00

高斯积分是 ∫−∞+∞e−x2dx\displaystyle \int_{-\infty}^{+\infty}e^{-x^2}\text{d}x∫−∞+∞​e−x2dx 。

学过微积分的都知道，∫e−x2dx\displaystyle \int e^{-x^2}\text{d}x∫e−x2dx 在初等函数范围内是不可积的。更何况高斯积分的积分上下界是 +∞+\infty+∞ 和 −∞-\infty−∞。这就决定了我们无法用微积分基本定理去求解。

但是，有数学家给出了天才的解法。如果不是吃透了定积分的定义，再加上神来之笔的灵感，是想不出这个解法的。我等后人只有膜拜了。

【解析】

令 I=∫−∞+∞e−x2dx\displaystyle I = \int_{-\infty}^{+\infty}e^{-x^2}\text{d}xI=∫−∞+∞​e−x2dx 。

定积分是一个数，与被积函数的自变量无关，

所以，换一下元，有 I=∫−∞+∞e−y2dy\displaystyle I = \int_{-\infty}^{+\infty}e^{-y^2}\text{d}yI=∫−∞+∞​e−y2dy 。

这样，凭空多出一个自变量 yyy，

二者相乘，得 I2=∫−∞+∞∫−∞+∞e−(x+y)2dxdy\displaystyle I^2 = \int_{-\infty}^{+\infty}\int_{-\infty}^{+\infty}e^{-(x+y)^2}\text{d}x\text{d}yI2=∫−∞+∞​∫−∞+∞​e−(x+y)2dxdy 。

二重积分可以用极坐标换元便于求解，

令 x2+y2=r2,dxdy=rdrdθ\displaystyle x^2+y^2=r^2, \text{d}x\text{d}y=r\text{d}r\text{d}\thetax2+y2=r2,dxdy=rdrdθ，其中 r∈[0,+∞),θ∈[0,2π]\displaystyle r\in\left[0,+\infty\right), \theta\in\left[0,2\pi\right]r∈[0,+∞),θ∈[0,2π]。

有 I2=∫02π∫0+∞e−r2rdrdθ\displaystyle I^2 = \int_{0}^{2\pi}\int_{0}^{+\infty}e^{-r^2}r\text{d}r\text{d}\thetaI2=∫02π​∫0+∞​e−r2rdrdθ

二重积分可以看做两个一元积分的乘法。既然被积函数没有 θ\thetaθ 这个自变量，可以把它先分离出来，

所以有 I2=∫02πdθ∫0+∞e−r2rdr=2π∫0+∞e−r2rdr\displaystyle I^2 = \int_{0}^{2\pi}\text{d}\theta\int_{0}^{+\infty}e^{-r^2}r\text{d}r = 2\pi\int_{0}^{+\infty}e^{-r^2}r\text{d}rI2=∫02π​dθ∫0+∞​e−r2rdr=2π∫0+∞​e−r2rdr。

被积函数多了一个 rrr，就可以凑微分了，然后根据微积分基本定理求定积分，

有 I2=−12×2π∫0+∞e−r2d(−r2)=−π⋅e−r2∣0+∞=π\displaystyle I^2 = -\frac{1}{2}\times 2\pi\int_{0}^{+\infty}e^{-r^2}\text{d}(-r^2) = -\pi\cdot e^{-r^2}\bigg|_0^{+\infty}=\piI2=−21​×2π∫0+∞​e−r2d(−r2)=−π⋅e−r2​0+∞​=π，

开方即得，I=π\displaystyle I=\sqrt{\pi}I=π​。

为什么 dxdy = rdrdθ？

参考了这个知乎答案 https://www.zhihu.com/question/368888687/answer/1447917992 。

有一个假设是，同样是面积微元 dσ\text{d}\sigmadσ，在直角坐标系和极坐标系下是一样大小的，只是计算方式不同。

直角坐标系：dσ=dxdy\displaystyle \text{d}\sigma=\text{d}x\text{d}ydσ=dxdy 。

极坐标系：

dσ=S扇M2OM3−S扇M1OM4=12(r+Δr)2⋅Δθ−12r2⋅Δθ=12[r2+2rΔr+(Δr)2−r2]Δθ=12[2rΔr+(Δr)2]⋅Δθ\begin{aligned} \text{d}\sigma&=S_{扇M_2OM_3}-S_{扇M_1OM_4}\\&=\frac{1}{2}\left(r+\Delta r\right)^2\cdot\Delta\theta-\frac{1}{2}r^2\cdot\Delta\theta\\&=\frac{1}{2}\left[r^2+2r\Delta r+\left(\Delta r\right)^2-r^2\right]\Delta\theta\\&=\frac{1}{2}\left[2r\Delta r+\left(\Delta r\right)^2\right]\cdot\Delta\theta \end{aligned} dσ​=S扇M2​OM3​​−S扇M1​OM4​​=21​(r+Δr)2⋅Δθ−21​r2⋅Δθ=21​[r2+2rΔr+(Δr)2−r2]Δθ=21​[2rΔr+(Δr)2]⋅Δθ​

由于 dθ≈Δθ\displaystyle \text{d}\theta\approx\Delta\thetadθ≈Δθ，(Δr)2\displaystyle \left(\Delta r\right)^2(Δr)2 是 2rΔr2r\Delta r2rΔr 的高阶无穷小（可以忽略），dr≈Δr\displaystyle \text{d}r\approx\Delta rdr≈Δr，

因此 dσ=12(2rΔr)⋅Δθ=rdrdθ\displaystyle \text{d}\sigma=\frac{1}{2}\left(2r\Delta r\right)\cdot\Delta\theta=r\text{d}r\text{d}\thetadσ=21​(2rΔr)⋅Δθ=rdrdθ。

综上所述 dσ=dxdy=rdrdθ\displaystyle \text{d}\sigma=\text{d}x\text{d}y=r\text{d}r\text{d}\thetadσ=dxdy=rdrdθ

巴哈姆特动画疯视频一直加载该怎么办?

本文最后更新于 2024-12-27T14:56:34+08:00

我用 Firefox 火狐浏览器上巴哈姆特动画疯看动画，但是视频一直加载不出来。怎么办？

经过摸索，解决方案如下。

  1. 科学上网。动画疯只允许台湾地区和港澳地区的用户观看，如果 IP 不在这三地，则无法观看。需要配置科学上网的代理，使用台湾、香港、或者澳门的节点连接动画疯。

  2. 广告系统被拦截。动画疯要求免费用户先看广告才能看视频。所以需要把以下域名加入科学上网代理的白名单，以及广告拦截插件（例如 uBlock、AdBlock）的白名单。巴哈姆特的广告基于谷歌广告。

    与巴哈姆特自身功能有关的域名：

    与谷歌广告有关的域名

  3. 配置 DNS。建议使用阿里 DNS，233.5.5.5 和 233.6.6.6。如果用阿里 DNS 还不行，需要另外配置 DoH（DNS-over-https）。

  4. 关闭浏览器的硬件加速功能，需要重启浏览器才生效。
